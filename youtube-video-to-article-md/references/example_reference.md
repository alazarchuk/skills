# Reference Documentation for Youtube Video to Article Markdown

This document provides guidance on transforming YouTube transcripts into high-quality articles.

## Article Structure Best Practices

### Narrative Synthesis
Instead of literal transcription, synthesize the spoken content into a professional written voice.
- **Hook:** Use the video's intro to create a compelling opening.
- **Sectioning:** Group related ideas from the transcript under descriptive subheadings.
- **Flow:** Use transition sentences between paragraphs to maintain narrative continuity.

### Technical Detail
- Ensure technical terms are spelled correctly by cross-referencing with the video description or common industry terms.
- For video essays, maintain the original author's tone and primary arguments while providing clear summaries.

### Formatting
- Use bold text for emphasis.
- Use blockquotes for direct, high-impact quotes from the speaker.
- Ensure all resources and links from the video description are preserved in a dedicated section.
